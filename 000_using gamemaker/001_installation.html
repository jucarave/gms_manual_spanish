Instalación y Requisitos del Sistema

Instalando GameMaker: Studio para uso personal.

Si tu versión de GameMaker: Studio no ha sido comprada a traves de Steam, vas a necesitar ejecutar el programa GMStudio-Installer.exe y seguir las instrucciones para instalarlo. Tienes las opción de instalar el programa en cualquier llugar que usted lo desee pero es recomendado que siga con las opciones predeterminadas. Una vez la instalación este completa, en el menú de inicio encontrarás un nuevo grupo de programas desde donde puedes iniciar GameMaker: Studio.

NOTA: Si usted compró GameMaker: Studio a traves de Steam, el programa debería instalarse automaticamente, dandote la opción de crear un icono en el menú de incio y en el escritorio. Cualquiera de estos pueden ser usados para ejecutar el programa en su computador, ó usted puede ejecutarlo desde la librería de Steam directamente.

Requisitos Básicos

Cualquier computador que tenga menos de cinco años debería ser capaz de ejecutar GameMaker: Studio sin ningún problema, pero hay unas especificaciones minimas que su PC debe tener para usar el programa, y hay especificaciones adicionales para los módulos que usted tenga.

IMPORTANTE: Los SDK y OS requeridos para cada plataforma pueden ser diferentes dependiendo de la fecha debido a los cambios hechos entre actualizaciones de GameMaker. Usted siempre puede encontrar los ultimos requisitos en el helpdesk de YoYo Games aquí

Windows

GameMaker: Studio requiere un PC rasonablemente moderno que tenga Windows XP, Vista, 7, o posterior. Una tarjeta gráfica compatible con DirectX 9 (o superior). Requiere una resolución de pantalla de al menos 1024x768 y 65536 (16-bit) colores (pero preferiblemente una de 32-bits). También una tarjeta de sonido compatible con DirectX 9, o un chip de sonido integrado. Es siempre recomendado que usted se asegure de que tiene los drivers mas recientes instalados para su sistema.

GameMaker: Studio Requiere que DirectX versión 9.0 o posterior este instalado en su computador. (Usted puede descargar esta versión de DirectX desde el sitio de Microsoft: http://www.microsoft.com.) Cuando este diseñando y probando juegos, los requisitos de memoria pueden ser muy altos (al menos 128 MB y preferiblemente mas, pero esto depende del sistema operativo). Cuando se encuentre ejecutando juegos, los requisitos de memoria son menos severos y dependen mucho del tipo de juego así como de los recursos que fueron utilizados.

NOTA: Aunque la mayoria de PCs modernos se distribuyen con DirectX 10 y superior, esto no significa que DirectX 9 este presente en su maquina, y usted ¡debería instalarlo desde el link otorgado arriba!

Tenga en cuenta que los usuarios de Steam también tienen una plataforma adicional para compilar juegos la cual es Steam Workshop. Mientras crea y prueba su juego, este módulo funcionará exactamente como el módulo Standar de Windows, sin embargo cuando este creando el ejecutable final se te mostrará información adicional para subir su juego directamente a Steam. Mas información aquí.

Windows 8

La versión Profesional de GameMaker: Studio tiene las plataformas de Windows 8 modo Nativo y Javascript de modo gratis. Estas plataformas requieren el sistema operativo Windows 8 para ejecutarse (y para pruebas), y este puede ser el SO principal o un sub-SO que usted puede crear con Windows 7 (o con otro SO que usted elija), sin embargo usted no puede crear una aplicación final para Windows 8 por fuera de ese SO.

Usted también necesita asegurarse de que usted tiene instalado Microsoft .Net Framework 3.5 (.Net4 se encuentra instalado por defecto con Windows 8, pero puede que necesites instalar la versión anterior también) la cual esta disponible aquí, sin embargo cuando usted instala GameMaker: Studio en su Windows 8 esta tarea debería ser ejecutada de forma automatica.

Otra cosa que es necesaria es una copia de Microsoft Visual Studio la cual puede descargar aquí. Usted debe usar solamente VS 2012 (Las versiones Express están bien), así que incluso si usted tiene una versión vieja de Visual Studio, esta no funcionará con este módulo. Una versión de este producto que soporta la creación de apps de JavaScript para la tienda de Windows es esencial ya que la necesitaras para crear los archivos *.pfx los cuales son su Llaves de desarrollador y necesita subir sus apps y juegos a la tienda de Windows.

La plataforma de exportación de Windows 8 de GameMaker: Studio creara una app especificamente para el sistema operativo Windows 8, y será compatible con las características únicas que este SO ofrece al usuario. Sin embargo, si usted desea publicar su juego a traves de la tienda de Microsoft para Windows, usted debe tener una Licencia de Desarrollador Microsoft la cual puede ser creada aquí. Hay un pequeño costo involucrado en registrarse para este tipo de cuenta y usted necesitara una tarjeta de crédito y cuenta bancaria validas para poder monetizar cualquier aplicación subida a la tienda.

Mas información sobre la configuración del módulo de Windows 8 se encuentra en el YoYo Games Helpdesk.

Módulos Adicionales

Los módulos anteriores están incluidos para todas las versiones de GameMaker: Studio pero los siguientes módulos están solamente disponibles para aquellos que actualicén a la versión Profesional (Más información sobre las diferentes versiones de GameMaker: Studio se encuentra aquí).

Mac OS X

Para usar el módulo de exportación para Mac OS X, usted necesita tener un computador Mac con un chipset Intel y que ejecute el sistema operativo Mavericks (OSX 10.9) o superior. Usted también necesita descargar e instalar Xcode para la versión de su SO (Mínimamente la versión 6 de Xcode). Es generalmente recomendado que usted utilice el último OS X y el último Xcode (Puede encontrar información sobre la versión de su Mac haciendo clic en el logo de "Apple" en la parte superior izquierda y seleccionando la opción "Acerca de este Mac" del menú desplegable).

Información adicional sobre como configurar el módulo Mac OS X se encuentra en el YoYo Games Helpdesk.

Ubuntu (Linux)

El módulo de Linux esta diseñado para ser compatible con el sistema operativo Ubuntu, y aunque debería funcionar con otros sistemas Linux, solo se garantiza que funcione con las ultimas versiones de este popular SO Linux.

Mas información sobre como configurar el módulo de Ubuntu se encfuentra en el YoYo Games Helpdesk.

iOS

El módulo iOS de GameMaker: Studio requiere que usted tenga un computador Mac (o al menos acceso a uno) y que tenga los requisitos minimos para el módulo de exportación MacOSX (ver arriba) y usted debe ser un desarrollador registrado Apple con todos los certificados y licencias vigentes. Para más información sobre como ser un desarrollador registrado Apple y como obtener los certificados necesarios, visite el sitio de desarrolladores de Apple aquí.

Cerca de todos los dispositivos iOS son compatibles con GameMaker: Studio incluyendo:
  • Iphone 4, 4S, 5, 5C, 5S, 6 y 6plus
  • iPad 1 y superior
  • iPod Touch de la 3ra genración en adelante

Tampoco importa si usted esta probando en un iPhone o en un iPad ya que todos los builds creados por GameMaker: Studio son, por estandar, aplicaciones universales. Esto significa que pueden ejecutarse en iPhones y iPads sin ningún problema.

Más información sobre como configurar el módulo de iOS se encuentra en el YoYo Games Helpdesk.

Android y Amazon Fire

Antes de comenzar a trabajar directamente con GameMaker: Studio y Android o Amazon Fire, es importante tener en cuenta que los dispositivos que van a ejecutar su juego varian inmensamente en capacidad. El ejecutable de GameMaker: Studio solo funcionará con Android 2.3(Gingerbread) y superior (API Level 9 y superior para el SDK de Android), lo cual cubre la mayor parte pero no todos los dispositivos que se existen actualmente, así que revise que versión de Android utiliza su dispositivo. También es recomendado que su dispositivo tenga un GPU con la versión de OpenGL 1.1 o superior. Dispositivos sin una GPU van a ejecutar sus proyectos de GameMaker: Studio, pero tendrán serios problemas de desempeño. La plataforma Amazon Fire soporta todos los dispositivos Fire TV, el Fire Stick y todas las tables Fire.

También necesita descargar el SDK de Android ya que GameMaker: Studio requiere algunos de los componentes de las diferentes versiones de las versiones Android y las herramientas que instala. Puede obtener una copia del SDK aquí. Si usted esta utilizando el YoYo Compiler (YYC) entonces también vas a necesitar instalar una copia del NDK de Android la cual se puede encontrar aquí.

Más información sobre como configurar el módulo de Android se encuentra en el YoYo Games Helpdesk.

HTML5

El módulo de exportación HTML5 requiere solamente un navegador que soporte HTML5. Tenga en cuenta que aunque los navegadores modernos soportan HTML5, no todos ellos soportan el estandar actual de WebGL, así que recomendamos Chrome para realizar las pruebas ya que este tiene todas las características soportadas por el módulo.

Más información sobre como configurar el módulo de HTML5 se encuentra en el YoYo Games Helpdesk.

Windows Phone

Exportar juegos utilizando el módulo de Windows Phone requiere que usted tenga instalado el SDK 8.0 de Windows Phone (descarguelo aquí ). También vas a necesitar un PC con Windows 8 con las siguientes especificaciones:
  • Windows 8 64-bits. Un sistema de 32-bits no esta soportado
  • 4GB de RAM mínimo.
  • 6.5GB de disco duro disponible

Si usted no tiene un dispositivo con Windows Phone, puede hacer uso de el emulador de Windows Phone, sin embargo tenga en cuenta que estos requisitos adicionales aplicarían en ese caso:

  • Windows 8 Profesional o Edición Empresarial
  • Virtualización de Hardware
  • Second Level Address Translation (SLAT)
  • DHPC, para que la máquina virtual pueda obtener una IP en la red local

NOTA: El SDK de Windows PHONE también instala Microsoft Visual Basic Express para Windows Phone, así que no hay necesidad de instalarlo por separado.

Más información sobre como configurar el módulo de Windows Phone se encuentra en el YoYo Games Helpdesk.

Tizen

El módulo de exportación Tizen tiene ciertos pre-requisitos que se deben cumplir antes de ser usado, el más importante de todos es que usted debe tener instalado el SDK de Tizen (Puede descargarlo aquí ).

¡ADVERTENCIA!: El SDK actual de Tizen No funciona en Windows 8, así que necesitas tener un sistema operativo anterior instalado para desarrollar (Puedes instalar varios sistemas operativos como dual boot). También tenga en cuenta que el SDK de 64-bit puede no funcionar, incluso si su SO es de 64-bit, así que si tiene dudas elija la versión de 32-bit.

A parte de eso también necesitas lo siguiente:
  • CPU con dual-core de 2 GHz
  • Al menos 2 GB de memoria RAM
  • Al menos 3 GB de espacio libre de disco duro
  • Oracle Java JDK (v6 o superior) instalado (no utilice OpenJDK)

Si usted no tiene acceso a un dispositivo Tizen, puede utilizar el Emulador Tizen, sin embargo también debe cumplir con los siguientes requisitos para que funcione correctamente:

  • Un CPU que soporte Intel VTx
  • Una resolución mínima de 1280 x 1024
  • Los drivers gráficos actualizados para OpenGL ES
Más información sobre como configurar el módulo Tizen se encuentra en el YoYo Games Helpdesk.

El YoYo Compiler (YYC)

El YYC toma el ejecutable normal de GameMaker: Studio y lo compila a código nativo para la plataforma específica, "Eliminando" funciones innecesarias y ejecutando técnicas de optimización para crear un ejecutable mejorado. Esto puede incrementar el desempeño dos o tres veces, especificamente en juegos con lógica pesada, y aumenta increiblemente el desempeño en general, ideal para aquellos juegos con usos intensivos de CPU.

El YYC es realmente un número separado de módulos, uno por cada plataforma disponible, y estará disponible para cualquier módulo activo, así que si usted tiene, por ejemplo, los módulos de iOS y Android, entonces veras dos nuevos modulos añadidos a su lista de plataformas disponibles - iOS (YYC) y Android (YYC). Si usted obtiene otros módulos en una fecha futura, el YYC estará disponible también mostrando un extra (YYC) junto al módulo de exportación.
NOTA: El YYC No está disponible para la plataforma HTML5, ni para ninguna otra que este basado en JavaScript (JS).

Los requisitos para el YYC dependen de las plataformas originales que usted posea y desee utilizar, ya que deben estar instaladas correctamente antes de hacer funcionar el YYC. Puede encontrar más información acerca de esto en el YoYo Games Knowledge Base al igual que en las páginas para la configuración individual de los módulos Global Game Settings.

El GameMaker: Player

Sin importar que versión de GameMaker: Studio usted posea, encontraras que viene con una aplicación separada incluida llamada El GameMaker: Player. Esta aplicación es una plataforma para distribuir sus juegos y para jugar aquellos juegos hechos por otros. Una vez tenga instalado GameMaker: Studio y el GameMaker: Player, puede ejecutarlos independientemente y también puede subir sus propios juegos para queu otros los jueguen.

Puede encontrar más información sobre el GameMaker: Player en el YoYo Games Knowledge Base - GameMaker: Player.
