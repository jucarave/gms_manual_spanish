Usando GameMaker: Studio

GameMaker: Studio es un programa fácil de usar para crear tus propios juegos de computadora.

GameMaker: Studio por defecto solo permite exportar juegos para Windows o el GameMaker: Player, mientras que la versión Professional también te permite compilar para Windows 8 (Del mismo modo que tiene funcionalidades adicionales). Si compraste GameMaker: Studio a traves de Steam, también tendrás un módulo adicional para exportar tus juegos al Steam Workshop. Puedes obtener modulos adicionales los cuales te permitirán exportar a otras plataformas como HTML5, Android, iOS, y Windows Phone 8:

  1. El módulo HTML5 te permite producir código HTML y Javascript listo para ejecutar que puedes incluir en cualquier sitio web
  2. El módulo Androd te permite crear y distribuir archivos *.apk
  3. Si tienes una cuenta de desarrollador iOS entonces con el módulo de iOS puedes publicar para iPad, iPhone y iPod
  4. El módulo de Windows Phone te permite apuntar a esos dispositivos para tus juegos (para dispositivos Windows Surface, utiliza el módulo de Windows 8)
  5. Con el módulo de Tizen puedes distribuir a un número vasto de dispositivos "inteligentes"
  6. Con el módulo de Mac OSX puedes distribuir a el market de Apple para escritorio
  7. El módulo de Linux te permite hacer juegos de escritorio que pueden ser ejecutados en el sistema operativo Ubuntu

Esos módulos están disponibles solo para aquellos que hayan actualizado a la versión Pro de GameMaker: Studio (más información sobre las diferentes versiones puede ser encontrada aquí).
