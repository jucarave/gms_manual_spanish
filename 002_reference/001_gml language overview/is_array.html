is_array

Valida si una variable es un array o no.

Sintaxis:

is_array(variable);


Argumento Descripción
variable La variable a validar.


Retorna: Boolean


Descripción

Esta función puede ser usada para validar si una variable es un array (en cuyo caso retornaría true) o no (en cuyo caso retornaría false).


Ejemplo:

if is_array(a)
   {
   a = 0;
   }

El código anterior validar si una variable es un array, y si lo es, entonces el array es eliminado.
