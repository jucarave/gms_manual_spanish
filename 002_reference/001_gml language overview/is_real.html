is_real

Retorna verdadero cuando el parametro pasado es un número real.

Sintaxis:

is_real(n);


Argumento Descripción
n El argumento para validar.


Retorna: Boolean


Descripción

Esta función retorna verdadero cuando la variable pasada es un número real o falso si no lo es. En algunos casos usted quiere validar si una variable en GameMaker: Studio contiene un número real, y en ese momento es cuando usted utilizaría esta función. Esta función no retorna el número real por el contrario retorna true o false, así que, si por ejemplo la variable tiene el valor "fish", la función va a retornar false, pero un valor 200 va a retornar true.


Ejemplo:

if is_real(val)
   {
   score += val;
   }

El código anterior valida que la variable "val" contenga un número real y si es así lo añade al puntaje.
