is_string

Retorna verdadero si el parámetro pasado es una cadena de texto.

Sintaxis:

is_string(n);


Argumento Descripción
n El argumento a validar.


Retorna: Boolean


Descripción

Esta función retorna verdadero cuando el tipo de variable pasado es una cadena de texto. En algunos casos usted va a querer validar si una variable en GameMaker: Studio contiene un string y no un real y ahí es cuando usted puede utilizar esta función. Esta función no retorna el string por el contrario retorna true o false, así que por ejemplo un valor "fish" va a retornar true mientras que un valor 200 va a retornar false.


Ejemplo:

if is_string(val)
   {
   name = "Player: " + val;
   }

El código anterior valida que la variable "val" sea un string y si lo es entonces lo añade a otro string el cual es asignado a la variable "name".
