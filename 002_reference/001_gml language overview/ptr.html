ptr

Trata de convertir un valor dado a un puntero.

Sintaxis:

ptr(n);


Argumento Descripción
n El valor a convertir.


Retorna: Puntero


Descripción

Esta función intentara convertir un valor pasado a un tipo de dato puntero, el valor debe ser ya sea un real, un string, un int64, un int32, o un ptr. Cualquier otro valor pasado va a causar que el juego colapse con un mensaje de error.


Ejemplo:

if !is_ptr(val)
   {
   ptr(application_surface);
   }

El código anterior revisara si la variable "val" es un puntero y si no lo es le asignará uno.
