async_load

Variable global que contiene un DS map el cual es usado en eventos asíncronos.

Sintaxis:

async_load;


Retorna: Real


Descripción

Esta variable tiene un alcance global y contiene un DS map cuando es usado en los Eventos Asíncronos y -1 cuando se usa en otro tipo de eventos. El contenido del DS map depende de el tipo de evento asíncrono que se ejecutó, revisa las páginas individuales sobre los eventos asíncronos para más detalle.


Ejemplo:

if ds_map_find_value(async_load, "id") == back
   {
   if ds_map_find_value(async_load, "status") >= 0
      {
      background_index[0] = back
      }
   }

El código anterior es tomado de una carga asíncrona de Imagenes, primero se revisa el ID del DS map del async_load, luego revisa el estado de la carga. Si el valor es mayor o igual a 0 entonces el resultado del llamado será utilizado como una nueva imágen en el background index del room actual.
