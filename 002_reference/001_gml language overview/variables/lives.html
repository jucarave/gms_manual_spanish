lives

Variable global para almacenar el número de vidas.

Sintaxis:

lives;


Retorna: Real


Descripción

Esta variable tiene un alcance global y contiene un valor número el cual se utiliza generalmente para controlar el número de vidas del jugador. Sin embargo este no es siempre el caso - Solo porque la variable se llame "lives" (vidas) no significa que deba usarse para controlar las vidas... Se puede usar para almacenar cualquier valor número útil para el juego.


Ejemplo:

if place_meeting(x, y, obj_Enemy_Bullet)
   {
   health -= 10;
   if health <= 0 lives -=1;
   }

El ejemplo anterior comprueba si hay una colisión con la instancia del objeto tipo "obj_Enemy_Bullet", y de ser así, remueve 10 puntos de salud al jugador, luego comprueba si este valor es igual o menor a 0. Si ese es el caso entonces se elimina 1 vida de la variable global "lives".
