Referencia

Esta sección contiene toda la información relacionada con el Game Maker Language

GameMaker: Studio contiene un lenguaje de programación propio, comunmente llamado GML, el cual le otorga mucha más flexibilidad y control sobre sus juegos. En esta sección describiremos el lenguaje GML y haremos una descripción de todas las funciones y variables disponibles para controlar todos los aspectos de su juego.
